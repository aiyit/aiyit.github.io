자료형. Data Type

파이썬이 데이터를 저장·처리하는 8가지 자료형 — 정수, 실수, 문자열, 논리형부터 리스트·튜플·딕셔너리·집합까지 실습 코드와 함께 정리한 2주차 강의 자료입니다.

01

자료형(Data Type)이란?

자료형(Data Type)이란 프로그램에서 데이터를 저장하고 처리하는 방식을 의미합니다. 컴퓨터는 모든 데이터를 0과 1의 이진수로 처리하지만, 프로그래밍 언어에서는 데이터를 효율적으로 다루기 위해 정수, 실수, 문자열, 논리값 등 다양한 자료형을 제공합니다.

나이
정수 (int)
키
실수 (float)
이름
문자열 (str)
합격 여부
논리형 (bool)
02

자료형 종류

자료의 종류에 따라 적절한 자료형을 선택합니다.

자료형의미예시
int정수10, -5, 1000
float실수3.14, -2.5
str문자열"Python", "AI"
bool논리형True, False
list리스트[10,20,30]
tuple튜플(10,20,30)
dict딕셔너리{"이름":"홍길동"}
set집합{1,2,3}

고급 과제 — 학생 정보관리 프로그램

딕셔너리·튜플·리스트·집합을 모두 활용해 한 학생의 정보를 종합적으로 관리·출력하는 프로그램입니다.

student_manager.py
# ============================================
# 학생 정보관리 프로그램
# 작성자 : AI소프트웨어과
# ============================================

# -----------------------------
# 기본 정보
# -----------------------------
student = {
    "학번": "20260001",
    "이름": "홍길동",
    "학과": "AI소프트웨어과",
    "학년": 1,
    "평점": 4.35,
    "장학금": True
}

# -----------------------------
# 튜플(생년월일)
# -----------------------------
birthday = (2007, 5, 12)

# -----------------------------
# 리스트(수강과목)
# -----------------------------
subjects = [
    "Python",
    "웹프로그래밍",
    "인공지능개론",
    "데이터분석"
]

# -----------------------------
# 집합(동아리)
# 중복 자동 제거
# -----------------------------
clubs = {
    "AI",
    "드론",
    "AI",
    "드론",
    "축구"
}

# -----------------------------
# 출력
# -----------------------------
print("="*45)
print("         학생 정보 관리 시스템")
print("="*45)

print(f"학번      : {student['학번']}")
print(f"이름      : {student['이름']}")
print(f"학과      : {student['학과']}")
print(f"학년      : {student['학년']}")
print(f"평점      : {student['평점']:.2f}")
print(f"장학금    : {student['장학금']}")

print("-"*45)

print(f"생년월일  : {birthday[0]}년 {birthday[1]}월 {birthday[2]}일")

print("-"*45)

print("수강 과목")

for subject in subjects:
    print("  ●", subject)

print("-"*45)

print("동아리")

for club in clubs:
    print("  ■", club)

print("="*45)

사용자 입력 — input()

키보드로 값을 입력받아 딕셔너리에 저장하고, .items()로 전체를 순회하며 정렬 출력합니다.

input_student.py
#사용자 입력
student = {}

student["학번"] = input("학번 : ")
student["이름"] = input("이름 : ")
student["학과"] = input("학과 : ")
student["학년"] = int(input("학년 : "))
student["평점"] = float(input("평점 : "))

print("\n" + "="*40)
print("입력된 학생 정보")
print("="*40)

for key, value in student.items():
    print(f"{key:8} : {value}")

print("="*40)
