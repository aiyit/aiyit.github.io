Block 요소와 Inline 요소

HTML 요소는 브라우저에서 차지하는 공간의 성질에 따라 크게 두 가지로 나뉩니다. 각 요소가 어떻게 배치되는지 시각적으로 이해해 봅니다.

□ Block 요소

  • 항상 새 줄에서 시작
  • 가로 폭 전체를 차지 (width: 100%)
  • width, height, margin, padding 모두 적용 가능
  • 내부에 block·inline 요소 포함 가능
  • 대표 태그: div, p, h1~h6, ul, table

— Inline 요소

  • 텍스트 흐름 안에 이어서 배치
  • 콘텐츠 너비만큼만 공간 차지
  • width·height 지정 불가, 좌우 margin·padding만 유효
  • 일반적으로 텍스트·다른 inline 요소만 포함
  • 대표 태그: span, a, strong, em, code

주요 Inline 요소들

Inline 요소는 텍스트 흐름 속에서 나란히 배치됩니다. 윤곽선은 실제 차지 영역입니다.

렌더링 결과
<span> 일반 텍스트 중에 span 요소 는 흐름을 유지하며 색이나 스타일을 지정하는 데 쓰입니다.
<a> 문서 본문 안에 하이퍼링크(a 요소) 를 삽입하면 줄이 바뀌지 않습니다.
<strong> / <em> 중요한 내용은 strong, 강조 표현은 em — 둘 다 inline이며 줄 안에 나란히 배치됩니다.
<code> 함수를 호출할 때 console.log("hello") 처럼 인라인 코드를 삽입할 수 있습니다.
<mark> / <abbr> / <small> 형광펜 표시, HTML, 그리고 작은 글씨 표기도 모두 인라인입니다.
<sub> / <sup> 화학식 H2O, 수식 x2+y2=r2 — 아래첨자·위첨자도 인라인 요소입니다.
비교
Block vs Inline 속성 비교
속성 Block 요소 Inline 요소
줄바꿈 항상 새 줄에서 시작 줄 내에 이어서 배치
기본 너비 부모의 100% 콘텐츠 너비만큼
width / height 적용 가능 ✔ 적용 불가 ✖
margin 4방향 모두 적용 좌·우만 유효
padding 4방향 모두 적용 좌·우 유효, 상·하는 주변 레이아웃에 영향 없음
하위 요소 block + inline 모두 포함 가능 inline 요소만 포함 권장
CSS 전환 display: inline / display: block / display: inline-block으로 변경 가능
실제 브라우저 렌더링 흐름 — 흰 배경이 실제 페이지입니다

Block: 제목이 한 줄을 전부 차지합니다

Block: <p>도 한 줄 전체를 점유합니다. 옆에 다른 요소가 올 수 없습니다.

Inline 요소들은 텍스트처럼 흐릅니다 — span으로 강조한 단어, 굵은 글씨, 기울임꼴, 링크가 모두 한 줄에 나란히 놓입니다.
