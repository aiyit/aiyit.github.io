함수. Function

필요할 때마다 불러 쓰는 프로그램의 부품, 함수 — 매개변수와 반환값부터 지역/전역 변수, 내장함수까지 정리한 5주차 강의 자료입니다.

01

함수(Function)란?

함수(Function)란 특정 작업을 수행하도록 만들어 놓은 코드의 묶음입니다. 쉽게 말하면 필요할 때마다 불러서 사용할 수 있는 프로그램의 부품(모듈)입니다. 예를 들어 학생의 평균을 계산하는 코드가 여러 곳에서 필요하다면, 매번 작성하는 대신 평균 계산 함수를 만들어 사용합니다.

02

함수를 사용하는 이유

함수를 사용하는 가장 큰 이유는 코드의 재사용성, 가독성, 유지보수성 향상입니다.

코드 재사용
같은 코드를 여러 번 작성하지 않음
가독성 향상
프로그램 구조를 이해하기 쉬움
유지보수 용이
한 곳만 수정하면 전체에 반영
모듈화
기능별로 나누어 개발 가능
협업
여러 사람이 기능별 개발 가능

함수 실습 — 학생 성적 종합

평균 계산 함수와 학점 판정 함수를 함께 사용해 사용자 입력 기반 성적표를 출력합니다.

grade_report.py
def average(scores):
    return sum(scores)/len(scores)

def grade(avg):

    if avg>=90:
        return "A"
    elif avg>=80:
        return "B"
    elif avg>=70:
        return "C"
    elif avg>=60:
        return "D"
    else:
        return "F"

name=input("이름 : ")

scores=[]

for i in range(3):
    score=int(input(f"{i+1}번째 점수 : "))
    scores.append(score)

avg=average(scores)

print("="*35)
print("학생 성적")
print("="*35)
print("이름 :",name)
print("평균 :",round(avg,2))
print("학점 :",grade(avg))
print("="*35)
