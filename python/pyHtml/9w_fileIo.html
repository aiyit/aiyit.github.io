📂 파이썬 파일 입출력 (File Input / Output)

1. 파일(File)이란?

파일(File)이란 데이터를 영구적으로 저장하는 공간입니다[cite: 2]. 프로그램이 종료되면 일반 변수는 모두 사라지지만, 파일에 저장한 데이터는 컴퓨터에 계속 남아 있습니다[cite: 2].

📌 대표적인 활용 예시
  • 학생 성적 저장[cite: 2]
  • 회원 정보 저장[cite: 2]
  • 시스템 로그 기록[cite: 2]
  • 환경 설정값 저장[cite: 2]

2. 파일 입출력 과정

파일 열기 (open) │ ▼ 읽기 (Read) 또는 쓰기 (Write) │ ▼ 파일 닫기 (close)
[cite: 2]

3. open() 함수

파일을 열 때 사용하는 기본 함수입니다[cite: 2].

# 파일객체 = open(파일명, 모드)
f = open("student.txt", "r")
[cite: 2]

4. 주요 파일 모드(Mode)

모드 의미 파일이 없을 경우
"r" 읽기 (Read) 오류 발생
"w" 쓰기 (Write) 새 파일 생성 (기존 내용 삭제)
"a" 추가 (Append) 새 파일 생성 (기존 내용 뒤에 이어 쓰기)
"x" 새 파일 생성 (Create) 이미 존재하면 오류 발생
"rb" 바이너리 읽기 이미지, 동영상 파일 등
"wb" 바이너리 쓰기 이미지, 동영상 파일 등
[cite: 2]

5. 파일 쓰기 (write)

f = open("output/hello.txt", "w")
f.write("안녕하세요.")
f.close()

# output 폴더로 이동하여 hello.txt 생성 여부를 확인합니다.

7. 파일 전체 읽기 (read)

12. with 문 사용하기 (권장)

with 구문을 사용하면 블록이 끝날 때 자동으로 파일이 닫히므로 f.close()를 명시하지 않아도 됩니다[cite: 2]. 실무에서 권장하는 방식입니다[cite: 2].

with open("output/student.txt", "r") as f:
    text = f.read()
    print(text)
[cite: 2]

13. UTF-8 인코딩 처리

한글 데이터를 저장하거나 읽을 때는 인코딩을 encoding="utf-8"로 지정하는 것이 안전합니다[cite: 2].

# 쓰기
with open("output/student.txt", "w", encoding="utf-8") as f:
    f.write("안녕하세요.")

# 읽기
with open("output/student.txt", "r", encoding="utf-8") as f:
    print(f.read())
[cite: 2]

17. 학생 성적 데이터 관리

students = [
    "홍길동,95",
    "김철수,88",
    "이영희,100"
]

# 파일 저장
with open("output/student.txt", "w", encoding="utf-8") as f:
    for student in students:
        f.write(student + "\n")

# 파일 읽기 및 출력
with open("output/student.txt", "r", encoding="utf-8") as f:
    for line in f:
        name, score = line.strip().split(",")
        print(name, score)
[cite: 2]

18. 파일 읽기 예외 처리

존재하지 않는 파일을 읽으려 할 때는 오류(FileNotFoundError)가 발생하므로 try-except 구문으로 예외 처리를 해줍니다[cite: 2].

try:
    with open("output/test.txt", "r", encoding="utf-8") as f:
        print(f.read())
except FileNotFoundError:
    print("파일이 없습니다.")
[cite: 2]

19. 자주 발생하는 오류 체크

  • 파일을 닫지 않음: close()를 누락하여 리소스가 낭비되는 경우. → with open() 사용 권장[cite: 2].
  • 잘못된 경로: 존재하지 않는 폴더 경로 지정 시 오류 발생 (예: open("abc/test.txt"))[cite: 2].
  • 인코딩 오류 (UnicodeDecodeError): 한글 깨짐 및 오류 방지를 위해 encoding="utf-8" 지정[cite: 2].

20. open() 주요 메서드 요약

함수 / 메서드 설명
open() 파일 열기
read() 파일 내용 전체를 문자열로 읽기
readline() 파일에서 한 줄씩 읽기
readlines() 파일의 모든 줄을 읽어 리스트로 반환
write() 파일에 내용 쓰기
writelines() 문자열 리스트를 파일에 쓰기
close() 열린 파일 닫기
[cite: 2]
