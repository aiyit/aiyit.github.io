반복문. for / while

같은 작업을 여러 번 반복 실행하는 반복문 — for와 range(), while, break/continue, 중첩 반복문, enumerate()까지 정리한 4주차 강의 자료입니다.

01

반복문이란?

반복문(Loop)이란 같은 작업을 여러 번 반복해서 실행하는 문법입니다. 동일하거나 유사한 작업을 반복 수행할 때 사용하며, 반복 횟수가 정해진 경우(for)와 조건이 만족하는 동안 반복하는 경우(while)로 나뉩니다.

학생 30명의 점수 출력
1부터 100까지의 합 계산
구구단 출력
파일의 모든 줄 읽기
리스트의 모든 요소 출력

for문과 range()

for_syntax.py
# for 변수 in 반복가능한객체:
#     실행문

# 반복 가능한 객체(Iterable)
# 문자열(str), 리스트(list), 튜플(tuple), 딕셔너리(dict), 집합(set), range()

range()는 일정한 범위의 숫자를 생성하는 함수입니다.

함수생성 값
range(5)0, 1, 2, 3, 4
range(1,6)1, 2, 3, 4, 5
range(2,11,2)2, 4, 6, 8, 10
range_demo.py
for i in range(5):
    print(i)

for i in range(1,6):
    print(i)

# while 조건:
#     실행문

continue — 현재 반복만 건너뛰고 다음 반복 실행
