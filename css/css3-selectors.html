핵심 선택자 6가지 — 문법, 설명, 실전 예제

01
Universal Selector
전체 선택자

문법

* { property: value; }

설명

별표(*)를 사용하며 HTML 문서 내의 모든 요소를 한 번에 선택합니다. 전역 리셋이나 공통 스타일 적용 시 주로 사용됩니다. 문서의 모든 태그에 영향을 주므로 성능에 주의가 필요합니다.

사용 사례

• box-sizing 전역 리셋
• margin / padding 초기화
• 디버깅용 아웃라인 표시

CSS HTML 
/* 모든 요소의 기본 여백 초기화 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* 디버깅: 모든 요소에 아웃라인 표시 */
* {
  outline: 1px dashed rgba(255,0,0,0.3);
}
미리보기 — 모든 요소에 아웃라인 표시
외부 div — span 텍스트 — 또 다른 span
다른 div 요소
02
Class Selector
클래스 선택자

문법

.className { property: value; }

설명

HTML 요소의 class 속성 값 앞에 점(.)을 붙여 선택합니다. 하나의 클래스를 여러 요소에 재사용할 수 있고, 한 요소에 여러 클래스를 동시에 지정하는 것도 가능합니다.

사용 사례

• 버튼, 카드 등 반복 컴포넌트
• 테마별 색상 변형
• 상태 클래스 (active, disabled)

CSS 
/* .highlight 클래스를 가진 요소만 선택 */
.highlight {
  background-color: #ffeb3b;
  padding: 4px 8px;
  border-radius: 4px;
  font-weight: bold;
}

/* p 태그 중 .highlight 클래스만 */
p.highlight {
  color: #b71c1c;
}
미리보기

일반 텍스트입니다. 스타일이 적용되지 않습니다.

.highlight 클래스가 적용된 텍스트

다시 일반 텍스트입니다.

이 요소에도 동일한 클래스 재사용 가능

03
ID Selector
ID 선택자

문법

#idName { property: value; }

설명

HTML 요소의 id 속성 값 앞에 샵(#)을 붙여 선택합니다. ID는 문서 내에서 고유해야 하므로 단 하나의 요소만 선택됩니다. 클래스보다 우선순위(specificity)가 높습니다.

사용 사례

• 페이지 내 유일한 레이아웃 영역
• 앵커 링크 타겟 (#section1)
• JavaScript DOM 접근 포인트

CSS HTML 
/* id="unique-box" 인 요소만 선택 */
#unique-box {
  background: linear-gradient(135deg, #e0f7fa, #b2ebf2);
  border: 2px solid #00acc1;
  border-radius: 8px;
  padding: 16px;
  text-align: center;
}
미리보기
🎯 이 박스만 #unique-box ID 선택자로 스타일이 적용됩니다
04
Child Combinator
자식 결합자 선택자

문법

parent > child { ... }

설명

꺾쇠(>)를 사용해 직접 자식 요소만 선택합니다. 후손(descendant) 선택자(공백)와 달리, 중첩된 하위 요소에는 스타일이 적용되지 않아 정밀한 스타일링이 가능합니다.

사용 사례

• 중첩 리스트의 1단계만 스타일링
• 탭 메뉴의 직속 링크 선택
• 중첩 컴포넌트 스타일 격리

CSS 
/* .menu의 직접 자식 li만 굵게 */
.menu > li {
  color: #f0a545;
  font-weight: bold;
}

/* 더 깊은 중첩 li는 별도 스타일 */
.menu > li > ul > li {
  color: #888;
  font-weight: normal;
}
미리보기 — 직접 자식만 강조색, 중첩 자식은 기본색
  • 메뉴 1 (직접 자식 → 강조)
    • 서브메뉴 1-1 (중첩 → 기본)
    • 서브메뉴 1-2 (중첩 → 기본)
  • 메뉴 2 (직접 자식 → 강조)
    • 서브메뉴 2-1 (중첩 → 기본)
05
Attribute Selector
속성 선택자

문법

[attribute="value"] { ... }

설명

대괄호([ ]) 안에 HTML 속성명과 값을 지정하여 선택합니다. 완전 일치 외에도 ^=(시작), $=(끝), *=(포함) 등 다양한 패턴 매칭이 지원됩니다.

사용 사례

• input 타입별 개별 스타일
• href가 https로 시작하는 링크
• data-* 커스텀 속성 기반 선택

CSS 
/* type="text" 인 input만 파란 테두리 */
input[type="text"] {
  border: 2px solid #5cb8f5;
  background: rgba(92,184,245,.1);
}

/* type="password" 는 초록 테두리 */
input[type="password"] {
  border: 2px solid #a8e06e;
  background: rgba(168,224,110,.1);
}

/* https로 시작하는 링크에 아이콘 추가 */
a[href^="https"]::after {
  content: " 🔒";
}
미리보기 — type 속성에 따라 다른 스타일
06
Pseudo-class Selector
가상 클래스 선택자

문법

selector:pseudo-class { ... }

설명

콜론(:)으로 요소의 특정 상태나 위치를 선택합니다. DOM에 없는 가상의 상태를 표현할 수 있습니다. :hover, :focus, :first-child, :nth-child() 등이 대표적입니다.

사용 사례

• :hover — 마우스 오버 상태
• :first-child / :last-child
• :nth-child(n) — n번째 요소
• :focus — 포커스 상태

CSS 
/* 첫 번째 자식 요소 */
li:first-child {
  color: #e05c9a;
  font-weight: bold;
}

/* 마지막 자식 요소 */
li:last-child {
  color: #7c6af5;
  font-weight: bold;
}

/* 짝수 번째 요소 */
li:nth-child(even) {
  background: rgba(255,255,255,.04);
}

/* 마우스 오버 */
a:hover {
  text-decoration: underline;
}
미리보기 — first-child(분홍), last-child(보라), even(배경)
  • 첫 번째 항목 :first-child → 분홍
  • 두 번째 항목 :nth-child(even) → 연한 배경
  • 세 번째 항목
  • 네 번째 항목 :nth-child(even) → 연한 배경
  • 다섯 번째 항목 :last-child → 보라

📋 선택자 요약 비교

# 선택자 문법 특이도(Specificity) 선택 범위
01 전체 선택자 * 0 모든 요소
02 클래스 선택자 .className 0,1,0 같은 클래스 다수 요소
03 ID 선택자 #idName 1,0,0 고유한 단일 요소
04 자식 결합자 A > B A+B 합산 직접 자식만
05 속성 선택자 [attr="val"] 0,1,0 속성 조건 일치 요소
06 가상 클래스 :pseudo 0,1,0 특정 상태·위치의 요소
