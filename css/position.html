CSS position 속성

static · relative · absolute · fixed · sticky — 5가지 배치 방식 예제

1. static

position: static

기본값. 요소를 HTML 문서 흐름 순서대로 배치합니다. top / right / bottom / left 속성이 적용되지 않습니다.

형제 요소 A
position: static (기본 흐름)
형제 요소 B

2. relative

position: relative

원래 있던 자리를 기준으로 top · left 등으로 이동합니다. 원래 자리는 그대로 유지(점선)되며 다른 요소에 영향을 주지 않습니다.

형제 요소 A
← 원래 위치 (공간 유지)
top: 16px · left: 24px 이동
형제 요소 B (밀리지 않음)

3. absolute

position: absolute

가장 가까운 position 지정 조상을 기준으로 배치됩니다 (없으면 <body>). 문서 흐름에서 완전히 빠져나와 다른 요소 위에 겹칩니다. 아래 주황 점선 박스가 position: relative 컨테이너입니다.

형제 요소 (absolute 요소에 의해 밀리지 않음)
컨테이너 (position: relative)
top:12px right:12px
bottom:10px left:12px

4. fixed

position: fixed

브라우저 뷰포트(화면)를 기준으로 고정됩니다. 스크롤해도 항상 같은 위치에 머뭅니다. 우측 하단의 빨간 배지가 실제 예시입니다.

📌 이 페이지 우측 하단에 고정된 빨간 배지 → position: fixed; bottom: 20px; right: 20px
스크롤해도 화면에 고정되어 있음을 확인하세요.
뷰포트 (스크롤 무관)
fixed 배지

5. sticky

position: sticky

평소에는 static처럼 흐름에 따라 배치되다가, 스크롤이 지정한 임계값(top: 0)에 도달하면 그 자리에 고정됩니다. 부모 요소 범위 내에서만 유지됩니다.

▌ 5가지 position 비교

값 기준점 문서 흐름 top/left 유효 주 용도
static 없음 (기본 흐름) 유지 ❌ 일반 요소 배치 (기본값)
relative 자기 원래 위치 유지 ✔ 미세 위치 조정, absolute 기준 컨테이너
absolute 가장 가까운 positioned 조상 이탈 ✔ 툴팁, 드롭다운, 오버레이
fixed 브라우저 뷰포트 이탈 ✔ 고정 헤더·푸터, 플로팅 버튼
sticky 스크롤 컨테이너 유지 → 고정 전환 ✔ (임계값) 테이블 헤더, 사이드바 메뉴
