In the era of the 4th Industrial Revolution, we systematically educate students with essential software fundamentals and practical AI technology through a combination of theory and practice. This department fosters practical professionals who can effectively utilize software and AI in various industrial fields.

The AI Software Department is a practice-oriented professional program designed to allow students to intensively acquire essential software fundamentals and AI technology in the era of the 4th Industrial Revolution within two years.

We provide practice-centered education on AI algorithms, data analysis, web/app development, and AI convergence service implementation skills required in a rapidly changing industrial environment, nurturing skilled professionals who can immediately be deployed into IT and AI fields after graduation.

In particular, this department, based on industry-academia cooperation with various companies, can lead to various career paths such as AI-based service companies, software developers, edutech, and IoT fields. We also support the acquisition of practical skills-based certifications and employment.

AI Web Programming

Web programming is the technology for developing web pages and web applications. It is an essential skill for creating dynamic web content that interacts with users, processing data, and implementing various functions.

  • HTML5 (HyperText Markup Language 5): A markup language that defines the structure and content of web pages. It allows various content such as text, images, and videos to be displayed on the web.
  • CSS3 (Cascading Style Sheets 3): A stylesheet language responsible for the design and layout of web pages. It controls visual elements like colors, fonts, spacing, and animations to make web pages beautiful.
  • JavaScript: A programming language that adds dynamic functionality to web pages. It is used to respond to user actions, manipulate data, and build complex web applications.
