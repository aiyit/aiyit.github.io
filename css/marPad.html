Margin & Padding

CSS 박스 모델에서 margin은 요소 바깥 여백, padding은 요소 안쪽 여백입니다.

박스 모델 구조

Content
Margin (바깥 여백) Border (테두리) Padding (안쪽 여백) Content (내용)

▌ Margin 예제 — 요소 바깥 공간

margin: 0 — 바깥 여백 없음

margin: 0

margin: 30px — 4방향 동일

margin: 30px

margin-top / bottom: 30px — 상하만

margin-top·bottom: 30px

margin: 0 auto — 가로 중앙 정렬 (width 필요)

margin: 0 auto

▌ Padding 예제 — 요소 안쪽 공간

padding: 0 — 안쪽 여백 없음

padding: 0

padding: 30px — 4방향 동일

padding: 30px

padding-top / bottom: 30px — 상하만

padding-top·bottom: 30px

padding-left / right: 40px — 좌우만

padding-left·right: 40px

▌ 단축 표기법 (Shorthand)

선언적용 순서
margin: 10px상·우·하·좌 모두 10px
margin: 10px 20px상하 10px / 좌우 20px
margin: 10px 20px 30px상 10px / 좌우 20px / 하 30px
margin: 10px 20px 30px 40px상 / 우 / 하 / 좌 (시계 방향)

padding도 동일한 규칙 적용
