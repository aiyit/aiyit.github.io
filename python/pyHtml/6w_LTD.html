컬렉션(Collection)이란?

컬렉션(Collection)이란 여러 개의 데이터를 하나의 변수에 저장하는 자료형입니다. 예를 들어 학생 30명의 이름을 저장하려면:

❌ 변수 30개
name1 = "홍길동"
name2 = "김철수"
name3 = "이영희"
...
⭕ 하나의 리스트
names = ["홍길동", "김철수", "이영희"]

리스트(List)

리스트는 여러 개의 데이터를 순서대로 저장하는 가장 많이 사용하는 자료형입니다. 순서가 있고, 인덱스(Index)를 가지며, 수정·추가·삭제·중복 저장이 모두 가능합니다.

생성
list_create.py
score = [90, 80, 100, 95]
fruit = ["사과", "배", "포도"]
mixed = [100, "Python", 3.14, True]

print(score)
인덱스(Index) — 0부터 시작
인덱스값
090
180
2100
395
list_index.py
score = [90,80,100,95]

print(score[0])
print(score[2])
print(score[-1])
수정 · 추가 · 삽입 · 삭제 · 길이
list_edit.py
#수정
score = [90,80,100]
score[1] = 85
print(score)

#추가
score.append(95)
print(score)

#삽입
score.insert(1,70)
print(score)

#삭제
score.remove(70)
del score[0]

#길이
print(len(score))
반복문 & 평균
list_loop_avg.py
for s in score:
    print(s)

#평균
average = sum(score)/len(score)
print(average)

튜플(Tuple)

튜플은 리스트와 거의 같지만, 한 번 생성되면 수정할 수 없는 자료형입니다.

생성 & 접근
tuple_create.py
color = ("Red","Green","Blue")
print(color)

print(color[0])
print(color[2])
⚠️ color[0] = "Black" → TypeError 발생 (튜플은 수정 불가)
반복
tuple_loop.py
for c in color:
    print(c)
💡 튜플을 사용하는 이유: 변경되면 안 되는 데이터를 저장하기 위함입니다. 예) 주민등록번호, 생년월일, GPS 좌표, RGB 색상
tuple_example.py
birthday = (2006,5,18)
print(birthday)

딕셔너리(Dictionary)

딕셔너리는 Key(키)와 Value(값)를 한 쌍으로 저장하는 자료형입니다. 실생활의 사전처럼 단어(Key)를 찾으면 값(Value)을 얻을 수 있습니다.

dict_basic.py
#생성
student = {
    "이름":"홍길동",
    "학과":"AI소프트웨어과",
    "학년":1,
    "평점":4.3
}
print(student)
print(student["이름"])

#추가
student["전화"] = "010-1111-2222"
#수정
student["학년"] = 2
#삭제
del student["평점"]

#모든 Key
print(student.keys())
#모든 Value
print(student.values())

#모두 출력
for key, value in student.items():
    print(key, value)

리스트 안의 딕셔너리

실무에서 가장 많이 사용하는 구조입니다 — 여러 명의 학생 정보를 딕셔너리 목록으로 관리합니다.

list_of_dict.py
students = [
    {"이름":"홍길동","점수":95},
    {"이름":"김철수","점수":88},
    {"이름":"이영희","점수":100}
]
for student in students:

    print(student["이름"],student["점수"])

학생 관리 프로그램 — 종합예제

리스트 안의 딕셔너리 구조로 여러 학생의 과목별 점수를 저장하고, 각 학생의 평균을 계산해 출력합니다.

student_manager.py
students = [
    {"이름":"홍길동","Python":95,"AI":90},
    {"이름":"김철수","Python":88,"AI":84},
    {"이름":"이영희","Python":100,"AI":98}
]
print("="*40)
for student in students:
    average = (student["Python"] + student["AI"]) / 2
    print(student["이름"])
    print("Python :",student["Python"])
    print("AI :",student["AI"])
    print("평균 :",average)
    print("-"*40)
