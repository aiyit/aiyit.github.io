MULTI-LAYER PERCEPTRON

은닉층을 포함한 이 모델은 데이터를 고차원으로 해석하여 비선형 결정 경계를 형성합니다.

1. 은닉층 연산 (Hidden Layer)

입력 값 $(x, y, z)$에 가중치($w$)와 편향($b$)을 적용한 후 활성화 함수를 거칩니다.

$h_j = \sigma \left( \sum_{i=1}^{3} w_{ij}^{(1)}x_i + b_j^{(1)} \right)$
  • $x_i$: 입력 좌표 (x, y, z)
  • $w_{ij}^{(1)}$: 입력층과 은닉층 사이의 가중치
  • $\sigma$: 시그모이드 활성화 함수

2. 출력층 연산 (Output Layer)

은닉층의 출력값($h$)들을 다시 조합하여 최종 판별 값을 계산합니다.

$\hat{y} = \sigma \left( \sum_{j=1}^{4} w_{j}^{(2)}h_j + b^{(2)} \right)$

최종 값 $\hat{y}$이 0.5보다 크면 파란색, 작으면 빨간색으로 분류됩니다.

3. 활성화 함수 (Sigmoid)

비선형성을 부여하여 곡선 경계를 가능하게 합니다.

$\sigma(z) = \frac{1}{1 + e^{-z}}$

학습 속도를 조절하며 가중치 $w$가 최적의 값을 찾아갈 때, 위 공식들이 어떻게 실제 공간의 노란색 경계면을 변화시키는지 학생들에게 설명해 주세요.
