📊 학생 성적 및 학점 계산 함수

📌 기능 요약
  • average(a, b, c): 세 과목 점수의 평균값을 계산하여 반환합니다[cite: 4].
  • grade(avg): 평균 점수에 따라 학점(A, B, C)을 평가하여 반환합니다[cite: 4].

1. 파이썬 함수 정의 (Source Code)

def average(a, b, c):
    return (a + b + c) / 3

def grade(avg):
    if avg >= 90:
        return "A"
    elif avg >= 80:
        return "B"
    else:
        return "C"

2. 모듈 사용 예시 (Usage Example)

해당 함수들을 모듈 파일 또는 동일 파일에서 사용할 수 있는 실행 코드 예시입니다.

# 1. 과목별 점수 설정
python_score = 95
ai_score = 90
web_score = 88

# 2. 평균 계산 함수 호출
avg = average(python_score, ai_score, web_score)

# 3. 학점 평가 함수 호출
student_grade = grade(avg)

# 4. 결과 출력
print(f"평균 점수: {avg:.2f}")  # 평균 점수: 91.00
print(f"최종 학점: {student_grade}")  # 최종 학점: A
