📦 파이썬 모듈(Module)

1. 모듈(Module)이란?

모듈(Module)은 함수(Function), 클래스(Class), 변수 등을 하나의 파일(.py)에 모아 놓은 것입니다[cite: 3]. 자주 사용하는 코드를 재사용하기 위해 만든 파이썬 파일입니다[cite: 3].

자주 사용하는 기능(더하기, 평균 계산, 학생 관리, 원 넓이 계산 등)을 매번 작성하지 않고 하나의 파일에 저장한 뒤 필요한 곳에서 가져와 사용합니다[cite: 3].

math.py ┌───────────────────────────────┐ │ add() │ │ sub() │ │ average() │ │ circle_area() │ └───────────────────────────────┘ └─► 필요한 함수만 가져와 사용
[cite: 3]

2. 모듈을 사용하는 이유

  • 코드를 재사용할 수 있습니다[cite: 3].
  • 유지보수가 용이해집니다[cite: 3].
  • 프로그램을 여러 파일로 분리하여 구조화할 수 있습니다[cite: 3].
  • 팀 단위 협업이 편해집니다[cite: 3].

3. 모듈의 종류

종류 설명 및 예시
내장 모듈 파이썬에 기본 포함된 모듈 (math, random, os 등)
표준 라이브러리 파이썬 표준으로 제공되는 모듈 (datetime, csv, json 등)
외부 모듈 pip로 설치하여 사용하는 모듈 (numpy, pandas, matplotlib 등)
사용자 모듈 개발자가 직접 작성한 모듈 (mymath.py 등)
[cite: 3]

13. import의 다양한 방법

# 방법 1: 모듈 전체 불러오기
import math
print(math.pi)

# 방법 2: 특정 기능만 가져오기
from math import pi
print(pi)

# 방법 3: 모듈의 모든 기능 가져오기 (비권장: 이름 충돌 가능성)
from math import *
print(sqrt(16))

# 방법 4: 별칭(Alias) 붙이기 (자주 쓰임)
import math as m
print(m.pi)

    return (a + b + c) / 3

def grade(avg):
    if avg >= 90:
        return "A"
    elif avg >= 80:
        return "B"
    else:
        return "C"
# main.py에서 불러오기

print("평균:", avg)
print("학점:", student.grade(avg))
